# Landing-2

## What's this?
This is landing page coded from *.psd file. Carousel is generated via mustache.js.

## Technologies

- HTML
- SCSS
- Vanilla JS
- Carousel Flickity
- Mustache.js

## How to run

- Download project from my github and unpack
- cd into unpacked folder
- npm install
- npm run watch, to start developing.
